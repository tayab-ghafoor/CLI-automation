## Setup Required

### One-Time Setup:

1. **Install Google libraries:**
   ```bash
   pip install google-auth-oauthlib google-auth-httplib2 google-api-python-client
   ```

2. **Create Google Cloud credentials:**
   - Go to Google Cloud Console
   - Create project
   - Enable Google Drive API
   - Create Service Account
   - Download JSON key

3. **Place credentials file:**
   - Rename: `google_drive_credentials.json`
   - Location: `system_manager_cli/` folder

4. **Share Google Drive folder:**
   - Create folder: "System Backups" (or any name)
   - Share with service account email
   - Set permission to "Editor"

**Detailed Instructions:** See `GOOGLE_DRIVE_SETUP.md`

## Frequently Asked Questions

**Q: Do I have to use Google Drive?**
A: No! You can skip it when prompted. Local backups work without Google Drive.

**Q: Can I use different email addresses?**
A: Yes! You can use your login email or specify a different email for Google Drive.

**Q: What if I don't know how to set up Google credentials?**
A: Follow the detailed steps in `GOOGLE_DRIVE_SETUP.md` - it's step-by-step.

**Q: What if upload fails?**
A: Your local backup is safely created. Google Drive upload is optional and doesn't affect local backups.

**Q: Are my credentials safe?**
A: Yes! Credentials are stored in a JSON file only (not in database or code). Never commit this file to Git.
